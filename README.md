# 🏫 SchoolSync – Modern School Management System

**SchoolSync** is a powerful, modern, and scalable School Management System designed to simplify school administration, enhance communication, and improve academic workflow for students, teachers, parents, and administrators.

Built with a modern MERN-style frontend stack and Firebase backend, SchoolSync delivers a smooth, fast, and intuitive user experience for managing educational institutions digitally.

## ✨ Key Features

### 🔐 Authentication & Security
- Role-based authentication (Admin, Teacher, Student, Parent)
- Secure login & registration using Firebase Authentication
- Protected routes for each user role

### 📊 Smart Dashboard
- Role-specific dashboards
- Real-time stats (students, attendance, grades, notices)
- Clean data visualization

### 👩‍🏫 Student Management
- Add, edit, delete student profiles
- Store academic and personal information
- Profile image upload via Firebase Storage

### 👨‍🏫 Teacher Management
- Manage teacher profiles and assigned subjects
- Class allocation system

### 📅 Class & Schedule System
- Create classes and sections
- Manage timetables efficiently

### 📖 Attendance System
- Daily attendance marking system
- Attendance history & reports

### 📝 Exams & Results
- Create exams and assign marks
- Automatic grade calculation (optional)
- Generate report cards

### 📢 Announcements System
- School-wide or role-based announcements
- Real-time updates via Firestore

### 💬 Communication (Optional)
- Parent ↔ Teacher messaging system
- Internal communication support

### ☁️ Cloud Storage
- Upload images, documents, and files using Firebase Storage

### 🎨 Modern UI/UX
- Fully responsive design (mobile, tablet, desktop)
- Smooth animations with Framer Motion
- Clean UI with Tailwind CSS + DaisyUI

### 🍞 User Experience
- Toast notifications for actions
- Loading spinners for async operations
- Smooth navigation with React Router
